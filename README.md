# 💫 KM Portfolio

A modern, high-performance, and fully responsive personal portfolio website built using **Next.js**, **Tailwind CSS**, and **Framer Motion**.

Designed with a futuristic dark UI, smooth animations, and clean user experience to showcase projects, skills, and developer identity professionally.
